ATOMIC

Fedora Workstation 是給筆記型電腦、桌上型電腦使用的作業系統，盡心打造，簡單易用；提供完善的工具組給各類開發者與自造者運用。

Fedora Server 是強大又不失彈性的作業系統，內含最好、最新的資料中心科技。它讓你一手掌控所有的基礎設施與服務。

Fedora Atomic 提供 Linux-Docker-Kubernetes (LDK) 應用堆疊的最佳平臺。

Fedora 永遠讓任何人自由使用、自由修改、自由散布。Fedora 是由全球各地的朋友一同努力，結合成為 Fedora Project 社群一心打造而成，讓社會大眾使用。

需要其他 Fedora 選擇嗎？

Fedora 社群也有發行 ARM 映像檔、其他 Live Spin、還有為各種特殊需求量身定製的 Fedora 特別版。請前往 Fedora Spins 或 Fedora Labs 網站瀏覽這些版本。
